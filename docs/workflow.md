# Workflow

## Multi-Agent Execution Pipeline

1. User submits request
2. Planner Agent decomposes the task
3. Coding Agent simulates implementation
4. Reviewer Agent validates output
5. Final response is returned

## Agent Responsibilities

### Planner Agent
Task analysis and workflow planning.

### Coding Agent
Implementation simulation and execution flow.

### Reviewer Agent
Validation, consistency checks, and review.
